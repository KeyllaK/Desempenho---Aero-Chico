# Testes de desempenho

Este projeto implementa um sistema para capturar dados do motor em uma bancada de desempenho

## 🎯 Funcionalidades
- Capta os dados de corrente do motor
-Dados de tração
-Dados de Empuxo
-Dados de Temperatura 

# Teste de corrente 
- Ligar o sensor de corrente Acs758 em série com o circuito (Bateria e esc).
- Ligar o sensor na placa prototipada.

# Teste de tração

#Teste de temperatura

## 🛠️ Tecnologias Utilizadas

- Linguagem C
- Pico SDK
- VS Code
- Placa BitDogLab
- Sensor de corrente Acs758

## 📋 Pré-requisitos

- VS Code instalado
- Pico SDK configurado
- Git instalado

## 🚀 Como Executar

1. Clone o repositório:
```bash
git clone [https://github.com/KeyllaK/servo.git]
```

2. Configure o ambiente de desenvolvimento seguindo as instruções do Pico SDK

3. Abra o projeto no VS Code

4. Execute na placa durante o teste.
